# Petpet Gif Generator - 在线宠物抚摸动画GIF生成器

## 项目介绍

Petpet Gif Generator 是一个免费的在线工具，允许用户从任何图像创建流行的"pet-pet"动画GIF，也称为宠物抚摸GIF和头部轻拍GIF。这些动画展示了一只手亲切地拍打您图像中的主体，创造出可爱而健康的效果，非常适合社交媒体、消息应用和论坛反应。

## 功能特点

### 🐾 多种动画效果
- **宠物抚摸GIF**：展示一只温柔的手轻抚您图像中的主体，创造出温暖而充满爱的效果
- **头部轻拍GIF**：展示一只俏皮的手轻拍主体头部，传达认可、鼓励或简单的喜爱

### ✨ 核心功能
- **简单上传**：支持从本地文件或URL上传任何图像
- **实时预览**：立即查看动画效果，实时调整参数
- **自定义控制**：
  - 调整图像大小和位置
  - 控制动画速度（FPS）
  - 设置挤压效果（Squish）
  - 翻转图像方向
  - 精确调整模式
- **高质量导出**：生成流畅、专业的GIF动画
- **响应式设计**：完美适配桌面和移动设备

### 🔧 技术特性
- **客户端处理**：所有图像和GIF生成都在浏览器中完成，确保隐私和速度
- **高性能渲染**：使用Canvas API优化图像处理和动画渲染
- **先进的GIF引擎**：基于gif.js构建，支持高质量GIF生成
- **实时预览系统**：即时反馈所有调整

## 使用方法

### 1. 选择图像
- 点击上传区域或拖放文件从设备上传任何图像
- 支持所有常见图像格式（PNG、JPG、JPEG、GIF等）
- 正方形图像效果最佳，但任何图像都会自动调整

### 2. 自定义动画
- 使用控制界面调整：
  - **大小（Size）**：调整主体图像的缩放比例
  - **挤压（Squish）**：控制动画中的挤压效果强度
  - **速度（Speed）**：调整动画播放速度（2-60 FPS）
  - **翻转（Flip）**：水平翻转主体图像
  - **调整模式（Adjust Mode）**：精确调整主体图像的位置

### 3. 控制播放
- 播放/暂停动画
- 逐帧查看动画效果
- 重置所有设置到默认值

### 4. 导出GIF
- 点击"导出"按钮生成GIF动画
- 等待生成完成（进度实时显示）
- 下载生成的GIF文件并分享

## 技术栈

- **前端框架**：原生JavaScript (ES6+)
- **图像处理**：Canvas API
- **GIF生成**：gif.js
- **样式**：sanitize.css, 自定义CSS Grid布局
- **图标**：jam-icons
- **字体**：Balsamiq Sans, Work Sans (Google Fonts)

## 核心算法

### 动画系统
- 5帧循环动画，模拟真实的抚摸/轻拍动作
- 每帧包含精确的位置、大小和变形参数
- 支持自定义帧率和延迟设置

### 图像处理
- 图像自动缩放和缓存，提高性能
- 支持跨域图像加载
- 透明度处理优化（使用绿色背景作为透明通道）

### GIF生成
- 使用Web Worker并行处理多帧
- 优化的颜色处理和透明度转换
- 实时进度反馈
